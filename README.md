A comprehensive CLI toolkit for working with JSON Schema. Infer schemas from raw JSON, validate data, diff schemas, and generate TypeScript types instantly. Designed to streamline data-driven development.

---

## 🚀 Why JSON Schema Tools?

JSON Schema is the backbone of modern API development, but managing it can be complex. **JSON Schema Tools** provides a unified interface to handle the entire schema lifecycle—from discovery to type safety.

### Key Features:
- **Schema Inference:** Generate valid JSON Schema from any raw JSON object.
- **Type Generation:** Instantly convert JSON Schema to high-fidelity TypeScript interfaces.
- **Validation CLI:** Validate local JSON files against schemas with detailed error reporting.
- **Schema Diffing:** Compare two schemas to find breaking changes or structural updates.
- **Developer Productivity:** One tool to replace multiple fragmented utilities.

## 🛠️ Installation

```bash
# Clone the repository
git clone https://github.com/AhmarHussain/json-schema-tools.git

# Navigate to the directory
cd json-schema-tools

# Install dependencies
npm install
```

---

## 📖 Usage Examples

### 1. Generate Types from Schema
```bash
npx json-schema-to-typescript user.schema.json > user.d.ts
```

### 2. Infer Schema from JSON
```bash
# Using the built-in inference utility
node scripts/infer.js data.json
```
